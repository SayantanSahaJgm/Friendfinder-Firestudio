# **App Name**: Proximity

## Core Features:

- Proximity Scan: Scan for nearby users via Bluetooth and WiFi.
- Location-Based Map View: Show friends and nearby users on a map using their location.
- Random Chat: Connect users in random, Omegle-style text, audio, and video chats.
- Main Feed: Display a feed of posts from friends, including text, media, likes, and comments.
- Story Sharing: Allow users to post stories that disappear after 24 hours.
- Profile Management: Enable users to create and edit their profiles.
- Smart Suggestion: Analyze a user's interests and suggest them topics or new users based on interests. The LLM acts as a tool by determining when and if an interest is popular enough to merit highlighting to the user, filtering out irrelevant suggestions and prioritizing connections based on shared passions.

## Style Guidelines:

- Primary color: Soft lavender (#E6E6FA) to create a calm and friendly atmosphere.
- Background color: Light off-white (#F8F8FF) for a clean, modern look.
- Accent color: Pale turquoise (#AFEEEE) for interactive elements.
- Body and headline font: 'PT Sans' (sans-serif) for clear readability and a touch of warmth.
- Use flat, minimalist icons for navigation and actions.
- Employ rounded cards and subtle shadows to create depth and soften the UI.
- Incorporate smooth, subtle animations for transitions and feedback.
